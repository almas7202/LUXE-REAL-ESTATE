# LUXE REAL ESTATE

A premium real estate website built with Next.js, React, and GSAP animations.

## Tech Stack

- **Framework:** Next.js 14.2.3
- **UI Library:** React 18
- **Styling:** TailwindCSS
- **Animations:** GSAP 3.12.5
- **Language:** TypeScript

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

### Build

```bash
npm run build
npm start
```
